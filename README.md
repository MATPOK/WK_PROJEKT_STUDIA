## Instalacja środowiska i zależności

Projekt posiada zautomatyzowany skrypt instalacyjny dla systemu Windows.
Wystarczy pobrać repozytorium i uruchomić jeden plik:

1. Pobierz i rozpakuj projekt.
2. Kliknij dwukrotnie plik `setup.bat` (lub uruchom go z poziomu terminala).
3. Skrypt automatycznie utworzy izolowane środowisko `venv`, pobierze wszystkie biblioteki z `requirements.txt` i podepnie je pod Jupyter Notebook.
